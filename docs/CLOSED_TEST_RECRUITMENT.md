# Closed-test tester recruitment

Google requires **12 testers opted in for 14 continuous days** before an *individual*
developer account can publish to production. Here's how to line them up fast.

## How it works (in Play Console)
1. **Testing → Closed testing → Create track** (or use the default "Alpha").
2. Add testers by **email list** (create one, paste the 12+ Gmail addresses) — testers
   must use the **same Google account** on their phone that you add here.
3. Copy the **"Join on the web"** opt-in link + the **Play Store install link** Console gives you.
4. Send testers the message below. They must (a) tap the opt-in link and accept, then
   (b) install from the Store link, and (c) **keep it installed for the full 14 days**.
5. Aim for **12–15 testers** (a couple always flake). The 14-day clock needs them opted in.

> Tip: it's the *opt-in* that counts toward the 14 days, so get everyone to tap the link on day 1.

---

## Message to send (chat / SMS / email)

> **Hey! Can you help me launch my weather app? 🌦️ (2 minutes)**
>
> I built a free Android app — **My Daily Weather** — and Google needs 12 testers for
> 2 weeks before I can publish it publicly. No catch, no cost.
>
> Could you:
> 1. Reply with the **Gmail address** on your Android phone (so I can add you).
> 2. After I add you, tap this link and accept: **[OPT-IN LINK]**
> 3. Install it here: **[PLAY STORE TEST LINK]**
> 4. Just keep it installed for ~2 weeks (open it whenever — feedback welcome!).
>
> That's it — you'd really be helping me get it live. 🙏 Thanks!

## Short version (for a group chat / Telegram / Viber)

> 🌦️ Launching my free weather app **My Daily Weather** and need **12 testers** for 2 weeks
> (Google's rule before it can go public). Drop your **Gmail** and I'll add you — then it's
> one tap to join + install, keep it ~14 days. Salamat! 🙏

## Where to find 12 fast
- Family + close friends with Android phones (quickest, most reliable).
- A barkada / family group chat (use the short version).
- Coworkers, or a local Telegram/Viber/FB group you're already in.
- Reddit r/androidapps "beta testers wanted" threads, or testing-swap communities
  (e.g. r/TestingAndroidApps) — people there test yours, you test theirs.
